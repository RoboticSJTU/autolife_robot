Bases: SpeakerInterface

Speaker implementation that handles both raw PCM and OPUS encoded audio.

This class implements the SpeakerInterface and provides functionality to: 1. Play PCM audio data with format conversion 2. Play OPUS encoded audio (if GStreamer is available) 3. Control volume and manage audio buffering 4. Handle speaker device initialization and recovery
