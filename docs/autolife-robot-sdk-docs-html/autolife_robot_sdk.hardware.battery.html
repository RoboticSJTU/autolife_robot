class autolife_robot_sdk.hardware.battery.rs485_reader.BatteryRS485Reader(port, baudrate, config=None)

Bases: BatteryInterface

Battery reader implementation using RS485 serial communication protocol. Provides methods to read various battery parameters such as voltage, current, capacity, temperature, and more.

Read battery data via RS485, parse it, and cache the results.

Args:

Returns:

Dictionary containing parsed battery data or None if read failed

restart()

Restart the battery reader by releasing and reinitializing resources

This method handles scenarios where: - The battery reader is currently running normally - The battery reader was previously closed with close() - The battery reader failed to initialize

Returns:

bool: True if battery reader was successfully restarted, False otherwise
